Features

Everything You Need.
One Binary.

No dependencies. No Python. No Node.js. Just one compiled Go binary that does it all.

Hardware Dashboard

Real-time battery, CPU, and network monitoring powered by termux-api. Live-updating gauges, color-coded status bars, and thermal readings — all in your terminal.

Battery % CPU Usage WiFi Info Thermal

Anti-Sleep Daemon

One-key toggle for termux-wake-lock — stop Android from killing your background processes. The #1 request from the Termux community.

Wake Lock PID Tracking Persistent

Network Toggles

Start and stop SSH and HTTP file servers from the TUI. Go-native net/http file server — no Python dependency required.

SSH Server File Server Port 8022 / 8081

IoT Webhook Bridge

HTTP webhook receiver that accepts sensor data from ESP32, Arduino, or any device. Persists to SQLite and displays in a scrollable live log.

HTTP POST SQLite Live Log MQTT Ready
Get Started

Up and Running
in 60 Seconds

Prerequisites: Termux + Go 1.22+ installed on your Android device.

1

Clone & Build

sh 
$ git clone https://github.com/Mr-Dark-debug/termnode.git
$ cd termnode
$ go build -o termnode .
2

Launch

sh 
$ ./termnode
3

Send IoT Data

sh 
$ curl -X POST http://localhost:8080/webhook/temperature \
  -d '{"sensor":"DHT22","value":23.5}'
{"status":"ok"}
