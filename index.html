Built for Power Users

Everything you need to manage your Android server, right from your pocket terminal.

📊

Hardware Dashboard

Real-time monitoring of battery health, CPU usage, and thermal states using Termux API.

⚙️

Daemon Manager

Control background processes and wake-locks to prevent Android from killing your services.

🌐

Network Controls

Toggle SSH, HTTP file servers, and view detailed network stats with zero configuration.

📡

IoT Bridge

Receive sensor data via HTTP webhooks and persist them to a local SQLite database.

Quick Installation

Get TermNode up and running on your device in seconds.

wget https://github.com/Mr-Dark-debug/termnode/releases/latest/download/termnode chmod +x termnode ./termnode

Or using Go:

go install github.com/Mr-Dark-debug/termnode@latest
